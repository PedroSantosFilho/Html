Listas HTML

As listas HTML permitem que os desenvolvedores da Web agrupem um conjunto de itens relacionados em listas.

Lista HTML não ordenada

Uma lista não ordenada começa com a tag <ul>li(unordered list) e termina na tag </ul> .

Os itens começa na tag <li> (list item) e termina na tag </li>

Os itens da lista serão marcados com marcadores (pequenos círculos pretos) por padrão:

   Exemplo
   <ul>
      <li>Coffe</li>
      <li>Tea</li>
      <li>Milk</li>
   </ul>

Resultado do programa acima

  • Coffe
  • Tea
  • Milk

Lista HTML Ordenada

Uma lista ordenada começa com a tag <ol>li(Ordered list) e termina na tag </ol>.

Os itens começa na tag <li> (list item) e termina na tag </li>

Os itens da lista serão ordenados por números por padrão:

   Exemplo
   <ol>
      <li>Coffe</li>
      <li>Tea</li>
      <li>Milk</li>
   </ol>

Resultado do programa acima

  1. Coffe
  2. Tea
  3. Milk

Listas de descrição HTML

HTML também suporta listas de descrição.

Uma lista de descrição é uma lista de termos, com uma descrição de cada termo.

A tag define a lista de descrição <dl> (define a description list), a tag define o termo (nome) <dt> (define a term in a description list) e a tag descreve cada termo <dd> (define a term in a description list): 

   Exemplo
   <dl>
   <dt>Coffe</dt>
      <dd> - black hot drink</dd>
   <dt>Tea</dt>
      <dd> - white cold drink</dd>
      </dl>

Resultado do programa acima

Café
- bebida quente preta
Chá
- bebida fria branca
