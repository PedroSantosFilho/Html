Formatação de texto HTML

HTML contém vários elementos para definir o texto com um significado especial.

Elementos de formatação HTML

Os elementos de formatação foram projetados para exibir tipos especiais de texto:

  • <b> - Texto em negrito
  • <strong> - Texto importante
  • <i> - Texto em itálico
  • <em> - Texto enfatizado
  • <mark> - Texto marcado
  • <small> - Texto menor
  • <del> - Texto excluído
  • <ins> - Texto inserido
  • <sub> - Texto subscrito
  • <sup> - Texto sobrescrito

Elementos HTML <b> e <strong>

O elemento HTML <b> define o texto em negrito, sem qualquer importância extra.

   Exemplo
   <p><b>Este texto está em negrito.</b></p>

Resultado do Programa Acima

Este texto está em negrito.

O elemento HTML <strong> define o texto com forte importância.
O conteúdo interno é normalmente exibido em negrito.

   Exemplo
   <p><strong>Este texto é importante</strong></p>

Resultado do Programa Acima.

Este texto é importante

Elementos HTML <i> e <em>

O elemento HTML <i>> define uma parte de texto em uma voz ou humor alternativo.
O conteúdo interno é normalmente exibido em itálico.

Nota: A tag é frequentemente usada para indicar um termo técnico, uma frase de outra língua, um pensamento, um nome de navio, etc.

   Exemplo
   <p><i>Este texto está em italico.</i></p>

Resultado do Exemplo Acima.

Este texto está em italico.

O elemento HTML <em> define texto enfatizado. O conteúdo interno é normalmente exibido em itálico.

Nota:Um leitor de tela pronunciará as palavras com ênfase.

   Exemplo
   <p><em>Este texto está enfatizado.</em></p>

Resultado do Exemplo Acima.

Este texto está enfatizado.

Elemento HTML <small>

O elemento HTML <small> define texto menor:

   Exemplo
   <p>Este é um texto normal.</p>
   <p><small>Este é um texto menor.</small></p>

Resultado do Exemplo Acima.

Este é um texto normal.

Este é um texto menor.

Elemento HTML<mark>

O elemento HTML <mark> define o texto, ou os textos que devem ser marcados ou destacados:

   Exemplo
   <p>Não se esqueça de comprar o <mark>leite</mark> hoje.</p>

Resultado do Exemplo Acima.

Não se esqueça de comprar o leite hoje.

Elemento HTML <del>

O elemento HTML <del> define o texto que foi excluído de um documento.

Os navegadores geralmente atingem uma linha através de texto suprimido:

   Exemplo
      <p>Minha cor bavorita é <del>azul</del> red.</p>

Resultado:

Minha cor bavorita é azul red.

Elemento HTML <ins>

O elemento HTML <ins> define um texto que foi inserido em um documento.

Os navegadores geralmente sublinham inserido Texto:

   Exemplo
   <p>Minha cor favorita é <del>azul</del> <ins>vermelha</ins>.</p>

Resultado:

Minha cor favorita é azul vermelha.
Azul ficou riscado e vermelha sublinhado.

Elemento HTML <sub>

O elemento HTML <sub> define texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal, e às vezes é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H2O:

   Exemplo
      <p>Este Texto será <sub> subscrito</sub>.

Resultado:

Este Texto será subscrito

Elemento HTML <sup>

O elemento HTML <sup> define texto sobrescrito. O texto sobrescrito aparece meio caractere acima do normal e, às vezes, é renderizado em uma fonte menor. Texto sobrescrito pode ser usado para notas de rodapé, como WWW[1]:

   Exemplo
      <p>Este é um texto <sup>subrescrito</sup>.</p>

Resultado:

Este é um texto subrescrito.
