Imagens HTML

As imagens podem melhorar o design e a aparência de uma página da web.

Inserindo paisagem no Html

Para inserir uma imagem no documento HTML usamos a tag <img>.

O atributo src é obrigátorio, ele mostra o endereço onde está localiza a imagem.

Imagens em outra pasta

Se você tiver suas imagens em uma subpasta, deverá incluir a pasta nome no atributo src .

Exemplo: src="../imagem/paisagens/paisage600.jpg"

O atributo alt também é obrigátorio, ele mostra o texto explicativo da imagem, caso ela não possa ser exibida.

Dica: um leitor de tela é um programa de software que lê o código HTML e permite que o usuário "ouça" o conteúdo. Leitores de tela são úteis para pessoas com deficiência visual ou dificuldade de aprendizado.

Observação: Na CSS Sempre especifique a largura (width) e a altura (height) de uma imagem. Se a largura e a altura não forem especificadas, a página da Web poderá piscar enquanto a imagem é carregada.

      <img src="../imagem/paisagens/paisage600.jpg" alt="paisagem" 
      width="500" height="400">

Inserindo uma Logomarca

O atributo src mostra o endereço onde está localiza a imagem.

O atributo alt mostra o texto explicativo para da imagem, caso ela não possa ser exibida.

         <img id="logo" src="../imagem/logomarcas/faetec-500.png" 
         alt="logomarca da Faetec">

Inserindo Logo Html:5

Inserindo a imagem da logomarca Html:5 de um pasta deferente.

O atributo src mostra o endereço onde está localiza a imagem.

O atributo alt mostra o texto explicativo para da imagem, caso ela não possa ser exibida.

          <img id="html" src="../imagem/logomarcas/html5-100.png" 
          alt="Logomarca HTML:5">

Inserindo Logo Css:3

Inserindo a imagem da logomarca Css:3 localizada na mesa pasta imagens.html.

O atributo src mostra o endereço onde está localiza a imagem.

O atributo alt mostra o texto explicativo para da imagem, caso ela não possa ser exibida.

         <img id="css" src="css3-100.jpg" alt="Logomarca Css:3">

Inserindo logomarca JavaScript

Inserindo a imagem da logomarca JavaScript localizada em outro site.

O atributo src mostra o endereço onde está localiza a imagem.

O atributo alt mostra o texto explicativo para da imagem, caso ela não possa ser exibida.

Imagens em outro servidor/site

Alguns sites apontam para uma imagem em outro servidor.

Para apontar para uma imagem em outro servidor, você deve especificar uma URL absoluta (completa) no atributo src :

         <img id="javascript" 
         src="https://logospng.org/wp-content/uploads/javascript.png" 
         alt="Logomarca JavaScript">

Nota: Caso o site deixe de existir, ou não possa ser exibido, só será mostrado o texto explicativo.

Observações sobre imagens externas: As imagens externas podem estar sob direitos autorais. Se não obtiver permissão para usá-lo, pode-se está violando as leis de direitos autorais. Além disso, não se pode controlar as imagens externas; elas podem ser removidos ou alterados repentinamente.

Imagem flutuante

Use a propriedade CSS float para deixar a imagem flutuar à direita float:right; ou à esquerda float:left; de um texto:

   Exemplo
   float:right; imagem para a direita.
   <p><img  src="../logomarcas/smiley.gif" alt="Smiley face" 
   style="float:right; width:42px;height:42px;">
   A imagem flutuará para a direita do texto.</p>

   float:left; imagem para a esquerda.
   <p><img src="../logomarcas/smiley.gif" alt="Smiley face" 
   style="float:left; width:42px;height:42px;">
   A imagem flutuará para a esquerda do texto.</p>

Resultado do Exemplo acima.

float:right; imagem para a direita.

A imagem flutuará para a direita do texto.

float:left; imagem para a esquerda.

A imagem flutuará para a esquerda do texto.

Resumo

  • Use o elemento HTML <img> para definir uma imagem
  • Use o atributo HTML src para definir a URL da imagem
  • Use o atributo HTML alt para definir um texto alternativo para uma imagem, se ela não puder ser exibida
  • Use os atributos HTML width (largura) e height (altura) ou as propriedades CSS width e height para definir o tamanho da imagem
  • Use a propriedade CSS float para deixar a imagem flutuar para a esquerda float:left; ou para a direita float:right;

Observação: Carregar imagens grandes leva tempo e pode deixar sua página da web mais lenta. Use as imagens com cuidado.
