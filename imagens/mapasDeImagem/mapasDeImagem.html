Mapas de Imagem HTML

Com mapas de imagem HTML, pode-se criar áreas clicáveis em uma imagem.

Mapas de imagem

A tag HTML <map> define um mapa de imagem. Um mapa de imagem é uma imagem com áreas clicáveis. As áreas são definidas com uma ou mais tags <area>.

Inserindo uma imagem

Para inserir uma imagem no documento HTML usamos a tag <img>.

O atributo src é obrigátorio, ele mostra o endereço onde está localiza a imagem.

O atributo alt também é obrigátorio, ele mostra o texto explicativo da imagem, caso ela não possa ser exibida.

   Exemplo
   <img src="/html/Html/imagens_1/outros/computador-celular-cafe.jpeg"
   alt="computador, celular e café">

Como funciona?

A ideia por trás de um mapa de imagem é que deve-se ser capaz de realizar diferentes ações, dependendo de onde se clicar na imagem.

Para criar um mapa de imagem, precisa-se de uma imagem e algum código HTML que descreva as áreas clicáveis.

A imagem

A única diferença de outras imagens é que você deve adicionar o atributo usemap: 

      Exemplo
      <img src="/html/Html/imagens_1/outros/computador-celular-cafe.jpeg"
      alt="computador, celular e café" 
      usemap="#workmap">

O valor usemap começa com uma marca de hash-tag # seguida pelo nome do mapa de imagem e é usado para criar um relacionamento entre a imagem e o mapa de imagem.

Dica: pode-se usar qualquer imagem como um mapa de imagem!

Criar mapa de imagem

Em seguida, adicione um elemento <map>. O elemento <map> é usado para criar um mapa de imagem e é vinculado à imagem usando o atributo obrigatório name="valor igual ao atributo usemap da tag img"

      Exemplo
      <img src="/html/Html/imagens_1/outros/computador-celular-cafe.jpeg"
      alt="computador, celular e café" 
      usemap="#workmap">
   
   <map name="workmap">

O atributo name deve ter o mesmo valor do atributo usemap da tag <img>.
