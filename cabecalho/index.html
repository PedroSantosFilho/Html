Cabeçalhos HTML

Títulos HTML são títulos ou legendas que você deseja exibir em uma página da Web.

Um título HTML ou tag HTML h pode ser definido como um título ou um subtítulo que você deseja exibir na página da Web.

Quando você coloca o texto dentro das tags de título <h1>......... </h1>, ele é exibido no navegador no formato em negrito e o tamanho do texto depende do número de cabeçalhos.

Existem seis cabeçalhos HTML diferentes que são definidos com as tags <h1> a <h6>, do nível mais alto h1 (título principal) ao nível mínimo h6 (título menos importante).

<h1> é a maior tag de cabeçalho e <h6> é a menor. Assim, h1 é usado para a posição mais importante e <h6> é usado para o mínimo importante.

Cabeçalhos em HTML ajuda o mecanismo de pesquisa a entender e indexar a estrutura da página da web.

Exemplos de Cabeçalhos HTML

      <h1>Cabeçalho 1</h1>
      <h2>Cabeçalho 2</h2>
      <h3>Cabeçalho 3/<h3>
      <h4>Cabeçalho 4</h4>
      <h5>Cabeçalho 5</h5>
      <h6>Cabeçalho 6</h6>
