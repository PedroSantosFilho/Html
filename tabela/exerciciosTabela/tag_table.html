Define Uma Tabela Html

Uma tabela em Html consiste de células dentro de linhas e colunas.

Cada célula da tabela é definida por uma tag de abertura <td> e uma tag de fechamento </tag>.

Exemplo de Tabela Simples SEM Borda

Tabela com text-align: center; alinhada ao centro sem bordas.

Nome Sobrenome Idade
Pedro Santos Filho 59
Cleonice Morais J. Santos 57
Tatiane Morais de J. Santos 32

Exemplo de Tabela Simples Com Borda

Tabela com text-align: center; o texto alinhado ao centro com bordas.

Para colocar bordas na tabela usa-se a propriedade CSS border dentro da tag Html style

Nome Sobrenome Idade
Pedro Santos Filho 59
Cleonice Morais de Jesus dos Santos 57
Tatiane Morais de J. Santos 32
