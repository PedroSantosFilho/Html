Tamanhos de tabela HTML

As tabelas HTML podem ter tamanhos diferentes para cada coluna, linha ou a tabela inteira.

Use o atributo com as propriedades ou para especificar o tamanho de uma tabela, linha ou coluna. width height

Largura da tabela HTML

Para definir a largura de uma tabela, adicione a propriedade CSS width ao elemento HTML tabel.

   table {
      background-color: white; /* Cor do fundo branca */
      color: black;    /* Cor das letras preta */
      width: 100%;     /* largura da tabela 100% */
   }

Nota: Usar uma porcentagem como a unidade de tamanho para uma largura significa quão largo esse elemento será comparado ao seu elemento pai, que neste caso é o elemento.<body>
