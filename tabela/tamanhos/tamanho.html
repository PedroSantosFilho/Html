Tamanhos de tabela HTML

As tabelas HTML podem ter tamanhos diferentes para cada coluna, linha ou a tabela inteira.

Use o atributo com as propriedades ou para especificar o tamanho de uma tabela, linha ou coluna. width height

Largura da tabela HTML

Para definir a largura de uma tabela, adicione a propriedade CSS width ao elemento HTML tabel.

   table {
      background-color: white; /* Cor do fundo branca */
      color: black;    /* Cor das letras preta */
      width: 100%;     /* largura da tabela 100% */
   }

Nota: Usar uma porcentagem como a unidade de tamanho para uma largura significa quão largo esse elemento será comparado ao seu elemento pai, que neste caso é o elemento.<body>

   Exemplo
   Uma tabela HTML simples:
   <table id="one">
      <tr>
         <th>Company</th>
         <th>Contact</th>
         <th>Country</th>
      </tr>
      <tr>
         <td>Alfreds Futterkiste</td>
         <td>Maria Anders</td>
         <td>Germany</td>
      </tr>
      <tr>
         <td>Centro comercial Moctezuma</td>
         <td>Francisco Chang</td>
         <td>Mexico</td>
      </tr>
   </table>

Resultado do Exemplo Acima

Company Contact Country
Alfreds Futterkiste Maria Anders Germany
Centro comercial Moctezuma Francisco Chang Mexico

Largura da Coluna da Tabela HTML

Para definir a largura de uma coluna específica, na CSS, no seletor th ou td use a propriedade width com o valor em x%

      Exemplo CSS
      th#two {
         width: 70%; /* LARGURA 70% */
      }
      Exemplo
      Uma tabela HTML simples:
      <table>
         <tr>
            <th>Company</th>
            <th id="two">Contact</th>
            <th>Country</th>
         </tr>
         <tr>
            <td>Alfreds Futterkiste</td>
            <td>Maria Anders</td>
            <td>Germany</td>
         </tr>
         <tr>
            <td>Centro comercial Moctezuma</td>
            <td>Francisco Chang</td>
            <td>Mexico</td>
         </tr>
      </table>

Resultado do Exemplo Acima

Company Contact Country
Alfreds Futterkiste Maria Anders Germany
Centro comercial Moctezuma Francisco Chang Mexico

Altura da Coluna da Tabela HTML

Para definir a altura de uma coluna específica, na CSS, no seletor tr, th ou td use a propriedade height com o valor em x%

         Exemplo CSS
         tr#three {
            height: 200px; /* altura da linha 40px */
         }
         Exemplo
         Uma tabela HTML simples:
         <table>
            <tr>
               <th>Company</th>
               <th>Contact</th>
               <th>Country</th>
            </tr>
            <tr id="three">
               <td>Alfreds Futterkiste</td>
               <td>Maria Anders</td>
               <td>Germany</td>
            </tr>
            <tr>
               <td>Centro comercial Moctezuma</td>
               <td>Francisco Chang</td>
               <td>Mexico</td>
            </tr>
         </table>

Resultado do Exemplo Acima

Company Contact Country
Alfreds Futterkiste Maria Anders Germany
Centro comercial Moctezuma Francisco Chang Mexico
