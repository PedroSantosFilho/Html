Tabela Html

Para criar uma tabela começamos pela tag <table></table>

Para criar uma linha na tabela usamos a tag <tr></tr> (table row)

Dentro da tag table row usamos a tag <th></th> (table head) para o nome do cabeçalho.

Dentro da tag table row usamos a tag <td></td> (table data) para os dados.

      Exemplo
      <table>
         <caption>HTml TABLE TAGS RESUmE</caption>
         <tr>
            <th>Tag</th>
            <th>Descriptian</th>
         </tr>
         <tr>
            <td><table></td>
            <td>it difines a table.</td>
         </tr>
         <tr>
            <td><tr></td>
            <td>it defines a row in a table.</td>
         </tr>
         <tr>
            <td><th></td>
            <td>is defines a heder cell in a table.</td>
         </tr>
         <tr>
            <td><td></td>
            <td>is defines a cell in a table.</td>
         </tr>
         <tr>
            <td><captino></td>
            <td>it difines the table caption.</td>
         </tr>
      </table>

Tabela HTML com Estilo CSS

Para adicionar uma borda na tabela, use a propriedade CSS border;

   table, th, td {
      border: 1px solid black;
   }

Para evitar bordas duplas, defina a propriedade CSS: border-collapse: como collapse

   table, th, td {
      border 1px solid black;
      border-collapse collapse;
   }

Se você definir uma cor de plano de fundo de cada célula, e dê à borda uma cor branca, você tem a impressão de uma borda invisível:

   table, th, td {
      border 1px solid white;
      border-collapse collapse;
   }
   th, td {
      background-color # 96D4D4;
   }

Com a propriedade CSS border-radius, as bordas ficam arredondadas pelos cantos:

   table, th, td {
      border 1px solid black;
      border-radius 10px;
   }

Com a propriedade border-style, você pode definir o aparência da borda.

   th, td {
      border-style dotted;
   }

Com a propriedade border-color, você pode definir a cor da borda.

   th, td {
      border-color # 96D4D4;
      }
