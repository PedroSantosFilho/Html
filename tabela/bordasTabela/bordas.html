Bordas Da Tabela HTML

As tabelas HTML podem ter bordas de estilos e formas diferentes.

Como adicionar uma borda

Ao adicionar uma borda a uma tabela, você também adiciona bordas ao redor de cada célula da tabela:

Para adicionar uma borda, use as propriedades CSS, e elementos:border, table, th, td

   Exemplo
      table, th, td {
         border 1px solid black;
      }

Bordas da tabela recolhidas

Para evitar bordas duplas, defina a propriedade CSS como.
border-collapse: collapse

Isso fará com que as bordas colapsem em uma única borda:

   Exemplo
   table, th, td {
      border 1px solid black;
      border-collapse collapse;
   }

Cor das Bordas da tabela

Se você definir uma cor de plano de fundo de cada célula, e dê à borda uma cor branca (a mesma que o plano de fundo do documento), você tem a impressão de uma borda invisível:

   Exemplo
   table, th, td {
      border 1px solid white;
      border-collapse collapse;
   }
   th, td {
      background-color #96D4D4;
   }

Aredondando As Bordas da Tabela

Com a propriedade border-radius, as bordas ficam arredondadas pelos cantos:

         
   Exemplo
   table, th, td {
      border : 1px solid black;
      border-radius : 10px;
   }

Para ignora a borda ao redor da tabela deixe de fora do seletor css:
table

         
   Exemplo
   th, td {
      border : 1px solid black;
      border-radius : 10px;
   }

Cor da borda

Com a propriedade border-color, você pode definir a cor da borda.

   Exemplo
    th, td {
      border-color: #96D4D4;
   }
