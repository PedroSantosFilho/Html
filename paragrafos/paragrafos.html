Parágrafo HTML

O parágrafo HTML ou a tag HTML <p> é usado para definir um parágrafo em uma página da Web.

Uma tag HTML <p> indica o início de um novo parágrafo.

Nota: Se estivermos usando várias tags <p> em um arquivo HTML, o navegador adicionará automaticamente uma única linha em branco entre os dois parágrafos.

   Veja este exemplo:
      <h2>Resultado do Programa Acima</h2>
      <p>Este é o primeiro parágrafo.</p>
      <p>Este é o segundo parágrafo.</p>
      <p>Este é o terceiro parágrafo.</p>

Resultado do Programa Acima

Este é o primeiro parágrafo.

Este é o segundo parágrafo.

Este é o terceiro parágrafo.

Espaço dentro do parágrafo HTML

Se colocar muitos espaços dentro da tag HTML <p>, o navegador removerá espaços extras e linhas extras durante a exibição da página. O navegador conta o número de espaços e linhas como um único.

      Exemplo:
      p>  
         Eu sou
         vai fornecer
         você um tutorial sobre HTML
         e espero que o faça
         ser muito benéfico para você.
      /p>
      p>  
         Olha, eu coloquei muito aqui
         de espaços, mas eu sei, Browser vai ignorá-lo.
      /p>
      p>  
         Não é possível determinar a exibição de HTML/p>
      p>porque as janelas redimensionadas podem criar resultados diferentes.
      /p>

Como usar a tag br> e hr> com parágrafo?

Uma tag HTML br> é usada para quebra de linha e pode ser usada com elementos de parágrafo. Veja a seguir o exemplo para mostrar como usar br> com o elemento p>.

   Exemplo:
      <h3> Uso de quebra de linha com tag <br></h3>
      <p><br>Papai e mamãe, e bebê e Dot,
      <br>Willie e eu?todo o lote
      <br>De todos nós fomos no trenó de Bimberlie,
      <br>Para a casa da avó no dia de Natal.</p>

Como usar a tag <hr> com parágrafo

Uma marca HTML <hr> é usada para aplicar uma linha horizontal entre duas instruções ou dois parágrafos. A seguir está o exemplo que está mostrando o uso da tag <hr> com parágrafo.

   Exemplo:
   <h2>Exemplo para mostrar uma linha horizontal com parágrafos</h2>
   <h2> Uma tag hr HTML desenha uma linha horizontal e separa dois parágrafos 
   com essa linha. <hr> ele começará um novo parágrafo.</p>

O elemento HTML <pre>

O elemento HTML <pre> define o texto pré-formatado.

O texto dentro de um elemento <pre> preserva é exibido em uma fonte de largura fixa (geralmente Courier), e espaços e quebras de linha:

<pre>
   Exemplo
      Minha Bonnie está sobre o oceano.
   
      Minha Bonnie está sobre o mar.
   
      Minha Bonnie está sobre o oceano.
   
      Oh, traga minha Bonnie de volta para mim.
   <pre>
