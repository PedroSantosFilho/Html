Citação HTML e elementos de citação

HTML <blockqoute> para citações

O elemento HTML <blockqoute> define uma seção que é citada de outra fonte.

Os navegadores geralmente indentam os elementos <blockqoute>.

   <p>Aqui está uma citação do site do WWF:</p>

   <blockqoute cite="http://www.worldwildlife.org/who/index.html">
   Por 60 anos, o WWF trabalhou para ajudar as pessoas e a natureza 
   a prosperar. Como organização de conservação líder mundial, 
   o WWF trabalha em quase 100 países. Em todos os níveis, 
   colaboramos com pessoas em todo o mundo para desenvolver 
   e fornecer soluções inovadoras que protegem as comunidades, 
   a vida selvagem e os lugares onde vivem.</blockqoute>

Resultado do Exemplo acima

Aqui está uma citação do site do WWF:

Por 60 anos, o WWF trabalhou para ajudar as pessoas e a natureza a prosperar. Como organização de conservação líder mundial, o WWF trabalha em quase 100 países. Em todos os níveis, colaboramos com pessoas em todo o mundo para desenvolver e fornecer soluções inovadoras que protegem as comunidades, a vida selvagem e os lugares onde vivem.

HTML <q> para citações curtas

A tag HTML <q> define uma citação curta.

Os navegadores normalmente inserem aspas ao redor da citação.

   Exemplo
   <p>O objetivo do WWF é: <q> Construir um futuro onde as pessoas 
   vivam em harmonia com a natureza.</q></p>

Resultado do Programa acima

O objetivo do WWF é: Construir um futuro onde as pessoas vivam em harmonia com a natureza.

HTML </abbr> para abreviaturas

A tag HTML </abbr> define uma abreviação ou um acrônimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".

As abreviações de marcação podem fornecer informações úteis para navegadores, sistemas de tradução e mecanismos de pesquisa.

Dica: Use o atributo de título global para mostrar a descrição da abreviação/acrônimo ao passar o mouse sobre o elemento.

   Exemplo
   <p>A <abbr> title="Organização Mundial da Saúde"> OMS </abbr> 
   foi fundada em 1948.</p>

Resultado do Programa acima

A OMS foi fundada em 1948.

HTML <address> para informações de contato

A tag HTML <address> define as informações de contato do autor/proprietário de um documento ou artigo.

As informações de contato podem ser um endereço de e-mail, URL, endereço físico, número de telefone, identificador de mídia social, etc.

O texto no elemento <address> geralmente é renderizado em itálico e os navegadores sempre adicionam uma quebra de linha antes e depois do elemento <address>.

   Exemplo
   <address>
      Escrito por John Doe.
      Visite-nos em:
      exemplo.com
      Caixa 564, Disneylândia
      EUA
   </address>

Resultado do Programa acima

Escrito por John Doe.
Visite-nos em:
exemplo.com
Caixa 564, Disneylândia
EUA

HTML <cite> para o título do trabalho

A tag HTML <cite> define o título de uma obra criativa (por exemplo, um livro, um poema, uma música, um filme, uma pintura, uma escultura, etc.).

Nota: O nome de uma pessoa não é o título de uma obra.

O texto no elemento <cite> geralmente é renderizado em itálico.

   Exemplo
   <p><cite>O Grito</cite> por Edvard Munch. Pintado em 1893.</p>

Resultado do Exemplo acima

O Grito por Edvard Munch. Pintado em 1893.

HTML <bdo> para substituição bidirecional

BDO significa Bi-Directional Override.

A tag HTML <bdo> é usada para substituir a direção do texto atual:

   Exemplo
   <bdo dir="rtl">Este texto será escrito da direita para a esquerda
   </bdo>

Resultado do Exemplo acima

Este texto será escrito da direita para a esquerda

Citação HTML e elementos de citação
Tag Description
<abbr> Defines an abbreviation or acronym
<address> Defines contact information for the author/owner of a document
<bdo> Defines the text direction
<blockquote> Defines a section that is quoted from another source
<cite> Defines the title of a work
<q> Defines a short inline quotation
