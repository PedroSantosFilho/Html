Atributos do Input HTML

O atributo de valor value

O atributo de entrada value especifica um valor inicial para um campo de entrada:

   Exemplo
   Campos de entrada com valores iniciais (padrão):
   <form>
      <label for="fname1">Primeiro nome:</label>
      <input type="text" id="fname1" name="fname" value="John">

      <label for="lname1">Último nome:</label>
      <input type="text" id="lname1"> name="lname" value="Doe">

      <input type="submit" value="Enviar"> 
   </form>

O atributo somente leitura readonly

O readonly atributo input, especifica que um campo de entrada é somente de leitura.

Um campo de entrada somente de leitura não pode ser modificado (no entanto, um usuário pode acessá-lo, realçá-lo e copiar o texto dele).

O valor de um campo de entrada somente leitura readonly será enviado ao enviar o formulário!

   
   Exemplo
   Um campo de entrada somente leitura:
   <form>
      <label for="fname">Primeiro nome:</label>
      <input type="text" id="fname" name="fname" value="John" 
            readonly>

      <label for="lname">Último nome:</label>
      <input type="text" id="lname"> name="lname" value="Doe">

      <input type="submit" value="Enviar">

   </form>

O atributo desabilitado disabled

O disabled atributo de entrada especifica que um campo de entrada deve ser desabilitado.

Um campo de entrada desabilitado é inutilizável e não clicável.

O valor de um campo de entrada desativado não será enviado ao enviar o formulário!

   
   Exemplo
   Um campo de entrada desabilitado:
   <form>
      <label for="fname2">Primeiro nome:</label>
      <input type="text" id="fname2" name="fname" value="John" 
            disabled>

      <label for="lname2">Último nome:</label>
      <input type="text" id="lname2"> name="lname" value="Doe">

      <input type="submit" value="Enviar">
   </form>

O atributo tamanho size

O size atributo de entrada especifica a largura visível, em caracteres, de um campo de entrada.

O valor padrão para sizeé 20.

Observação: o size atributo funciona com os seguintes tipos de entrada: texto, pesquisa, tel, url, email e senha.

   
   Exemplo
   Define uma largura para um campo de entrada:
   <form>
   <label for="fname4">Primeiro nome:</label>
   <input type="text" id="fname4" name="fname" value="John" 
         size="50">

   <label for="lname4">Último nome:</label>
   <input type="text" id="lname4"> name="lname" value="Doe" size="4">

   <input type="submit" value="Enviar">
   </form>
